Rapid e4473

Secure ASP .NET 4.51 Web App

This application consists of:

  • MVC ASP Web Application
  • Single Controller that contains all actions in order for 4473 form.
  • Theming using Bootstrap.
  • Native AES encryption algorythm for encryption / decryption on the fly.
  • BCrypt function for password encryption / comparison. (No method of retrieving plain txt psswd)
  • Separate Data Models in true MVC form.

Customize app

  • HTML, and TXT email templates saved in flat files in App_Data file within C:/Inetpub/wwwroot/Rapid4473.
  • XML document contains email host settings.

Deploy

  • Code is ready to be compiled for release in IIS directory.  NOT a flat file transfer.
  • SQL Server Table Creation - "Create_Rapid_e4473_Database.sql" included in project source code.
  • DB Entity Connection:
  • name="RAPID_e4473Entities"
  • connectionString="metadata=res://*/Models.e4473.csdl|res://*/Models.e4473.ssdl|res://*/Models.e4473.msl;provider=System.Data.SqlClient;provider connection string="data source=localhost;initial catalog=RAPID_e4473;persist security info=True;user id=XXX;password=*********;multipleactiveresultsets=True;application name=EntityFramework"" providerName="System.Data.EntityClient"
